Training Rules

Warm-Up

  • First move: 20→12→8→4 feeders
  • Work sets: Top set RPE 9-10 + back-offs

Intensifiers

  • DS Drop set final set only
  • RP Rest-pause: 10→5→3

Rest

  • Compounds: 90-150s
  • Legs: 120-180s
  • Isolation: 60-90s

Execution Tips

Incline Press

  • Angle: 30-35° not 45°
  • Touch: High on chest near clavicle
  • Lockout: Don't — keep tension

Lateral Raise

  • Weight: 15-20 lb max
  • Form: Lead with elbows, lean forward
  • Tempo: Pause 1s at top, no momentum

Lower Lat Pulldown

  • Pull to: Hip pocket, not chest
  • Setup: Single-arm, offset stance
  • Feel: Stretch top, squeeze bottom

Rear Delt Fly

  • Cue: Lead with pinkies
  • Control: No swing — go lighter

Cardio

  • Resistance: Low-moderate
  • HR: 120-135 bpm
  • Tip: Watch something

Posing (Daily 3-5 min)

  • Vacuums: 3×25-30s
  • Front double bi: Hold vacuum
  • Rear lat spread: Show lower lats
  • Side chest: Gut in, twist

If Recovery Tanks

  • Cut 1st: Saturday rear delts
  • Cut 2nd: Wednesday laterals
  • Cut 3rd: One back-off set per compound
