Training Rules

Warm-Up Protocol

  • First movement: 20→12→8→4 rep feeder sets
  • Work sets: Top set RPE 9-10 + back-off sets
  • Purpose: Prep joints, build to working weight safely

Intensity Techniques

  • DS Drop Set: Final set only — drop weight 20-30%, continue to failure
  • RP Rest-Pause: Hit failure, rest 10-15 sec, continue (10→5→3 reps)
  • Partials: After failure on full reps, do partial reps in stretched position

Rest Periods

  • Compounds: 90-150 seconds
  • Leg Compounds: 120-180 seconds
  • Isolation: 60-90 seconds
  • Giant Sets: No rest between exercises, 60-90s after round

Weak Point Priorities

  • Upper Chest: Incline 2×/week (Mon + Fri), low-to-high cables, 30-35° only
  • Side Delts: 3×/week across split, light weight, strict form
  • Rear Delts: 3×/week (Tue + Fri + Sat), high frequency brings them up
  • Lower Lats: Single-arm pulldowns to hip, straight-arm work 2×/week
  • Mid-Back: Wide-grip rows with elbows flared, squeeze shoulder blades

Recovery Management

  • If beat up, cut in order:
  • 1. Saturday session → make it rest day
  • 2. Wednesday laterals → hit Mon + Fri anyway
  • 3. One back-off set per compound → top sets still stimulate
