Training Notes

Warm-Up Protocol

  • First movement: 20 → 12-15 → 8-10 → 4-6 (feeders)
  • Work sets: 1 top set RPE 9-10 + 1-2 back-offs

Intensifiers

  • DS Drop set final set only (machines/cables)
  • RP Rest-pause: 10 reps → 10s → 5 → 10s → 3

Rest Periods

  • Compounds: 90-150 sec
  • Legs: 120-180 sec
  • Isolation: 60-90 sec

Progression

  • Add reps first within range
  • Increase load after hitting top of range 2x
