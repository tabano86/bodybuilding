Training Rules & Targets

Warm-Up Protocol

  • First movement only: 20 reps → 12-15 → 8-10 → 4-6 (feeders)
  • Then: 1 top set @ RPE 9-10 + 1-2 back-offs (-10-15% load)
  • Subsequent movements: 1-2 light feeders then work sets

Intensifiers

  • DS = Drop set on final set only (machines/cables)
  • RP = Rest-pause: 10 reps → 10s → 5 reps → 10s → 3 reps
  • Partials: After failure, 4-5 partial reps from stretched position

Rest Periods

  • Upper body: 90-150 seconds
  • Legs: 120-180 seconds
  • Pump/isolation: 60-90 seconds

Progression Rule

  • Add reps first within the range
  • Increase load only after hitting top of range 2 sessions in a row
  • During cut: maintaining strength = winning

Nutrition (Cut Phase)

  • Calories: ~2,700 kcal/day (starting)
  • Protein: 240g | Fat: 80g | Carbs: 170g
  • Adjustment: If no 0.6-1.2 lb drop for 2 weeks → reduce 200 kcal (carbs first)

Cardio Targets

  • Frequency: 4-6 LISS sessions/week (bike only)
  • Duration: 25-40 min standard | 15-30 min on leg day
  • Intensity: RPE 5-6 (can hold conversation)
  • Timing: Post-lift preferred, or split AM/PM

Daily Vacuum Protocol

  • Vacuums: 3 x 25-30 seconds (exhale fully, pull navel to spine)
  • Pose breathing: 3 x 10 seconds (ribs down, abs tight)
  • When: Morning, pre-workout, or before bed
  • Note: Limit loaded ab work to 2x/week to avoid thickening

Exercises to Avoid (Spine Safety)

  • Conventional deadlifts
  • Barbell back squats
  • Hack squats (machine)
  • Bulgarian split squats
  • Hip thrusts
  • Step-ups
  • Anything causing lumbar shear or compression
